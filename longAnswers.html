Proposed summer project.

CDK (Component Developer Kit) is a project i find very interesting seeing that it will add more flexibility to the system. Sometime, users specifically experienced one, are stuck during there development because a feature they want to use is unavailable as a component. Having a Component Developer Kit will allow them to build their apps more efficiently. Furthermore, this can be a starting point for an open plugin store where developers can submit created components. This will allow users to have more choices when selecting a component and will allow them to create more diverse apps.

However, this a task with multiple challenges seeing that it will increase the system complexity. All components created by users can work alone but interactions can sometimes cause a problem.

I like working on challenging project like this one because that allow me to gain additional skills and make me discover my real potential. Otherwise, i am open to another project if the mentor thinks that it will be more suitable for my profile.
